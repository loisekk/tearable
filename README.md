## Features

- **Verlet physics** ‚ÄĒ stable, accurate point-mass simulation with configurable damping
- **Gravity** ‚ÄĒ cloth naturally drapes and falls under gravitational force
- **Anchor points** ‚ÄĒ top row pinned at every 4th point to act as a curtain rail
- **Wind system** ‚ÄĒ sinusoidal oscillating wind force toggled on/off at runtime
- **Drag interaction** ‚ÄĒ attract cloth toward the cursor with left-click
- **Tear / cut mode** ‚ÄĒ right-click or switch to cut mode to slice through the fabric
- **Auto-tear** ‚ÄĒ sticks that stretch beyond 2.8√ó their rest length snap automatically
- **Tension coloring** ‚ÄĒ sticks shift from green ‚Üí yellow ‚Üí red based on current stretch
- **Boundary collisions** ‚ÄĒ points bounce off all four screen edges
- **On-screen HUD** ‚ÄĒ live FPS counter and control reference overlay

## How It Works

### Verlet Integration

Each point stores its current and previous position. Velocity is implicit ‚ÄĒ derived from the difference between the two. This makes the simulation naturally stable and energy-conserving.

```
velocity  = (current - previous) √ó damping
previous  = current
current  += velocity + acceleration
```

### Stick Constraints

Each connection between adjacent points enforces a rest length. On every frame, constraints are solved iteratively (5 passes) ‚ÄĒ the two endpoints are nudged apart or together to satisfy the target distance. Locked anchor points are immovable.

### Tearing

A stick is removed when its current length exceeds `REST_LENGTH √ó 2.8`. Mouse-based cutting removes sticks whose midpoint falls within a radius of the cursor.

### Tension Coloring

Stretch factor = `current_length / rest_length ‚ąí 1`. This is mapped `[0, 0.25]` ‚Üí green to red, giving instant visual feedback on stressed regions.

## Configuration

Key constants at the top of `cloth_simulation.py`:

| Constant | Default | Description |
|---|---|---|
| `COLS` | `40` | Number of cloth columns |
| `ROWS` | `25` | Number of cloth rows |
| `REST` | `18` | Rest length of each stick (px) |
| `GRAVITY` | `0.5` | Downward acceleration per frame |
| `DAMPING` | `0.99` | Velocity retention factor (0‚Äď1) |
| `ITERATIONS` | `5` | Constraint solving passes per frame |
| `TEAR_DIST` | `REST √ó 2.8` | Auto-tear threshold |
| `CUT_RADIUS` | `28` | Mouse cut influence radius (px) |
